Custom Properties 进阶

这是一个自定义的属性.

css:代码

              
                :root {
                  --main-color: red;
                }
                .custom-properties {
                  --main-color: rebeccapurple;
                  color: var(--main-color);
                }

全局root变量的颜色

在全局的 :root 选择器中定义的 Custom Properties 可以在整个文档中使用。

这就是为什么 Element-Plus 有的需要局部自定义宽高需要在 节点的 style 中直接设置 CSS 变量，而不是在全局的 :root 中设置，因为全局的 CSS 变量会被 Element-Plus 的样式覆盖掉。
